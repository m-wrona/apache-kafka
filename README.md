# apache-kafka

Modified samples from [Confluent](https://github.com/confluentinc/examples) in order to play with new features of Kafka a bit.
 
## Development

Pre-requisites:

* docker-compose

* Java

* Maven

1) Build project

```
mvn package
```

2) Run locally

```
docker-compose up
```

and then:

```
./run_web.sh
```
